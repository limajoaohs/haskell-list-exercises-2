# Haskell List Exercises

**Author:** João Henrique Silva Lima  
**University:** UFCG

## About
This repository contains solutions for the "Exercícios Haskell 2" list. The goal is to practice recursion, pattern matching, and list manipulation in Haskell.

## List of Solutions

* **q1.hs**: Calculate list length.
* **q2.hs**: Find element at index `i`.
* **q3.hs**: Find the largest number in a list.
* **q4.hs**: Keep only uppercase letters in a string.
* **q5.hs**: Add `n` spaces to the beginning of a string.
* **q6.hs**: Count words ending in 's' (plural).
* **q7.hs**: Sum symmetric elements (first + last).
* **q8.hs**: Unify (flatten) a list of lists.
* **q9.hs**: Split a list into two at index `n`.

## How to Run

1.  Open the terminal in this folder.
2.  Start GHC Interactive:
    ```bash
    ghci
    ```
3.  Load a file (e.g., Question 9):
    ```haskell
    :load q9.hs
    ```
4.  Run the function:
    ```haskell
    splitList [1, 2, 3, 4] 2
    ```
